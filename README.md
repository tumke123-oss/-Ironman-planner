# Ironman Hamburg Planner PWA

iPhone-ready web-app met:
- dagplanning t/m racedag
- schuifjes voor afstand, uren, zwemtempo, fietswattage en looptempo
- intervalblokken per sport
- automatische weektotalen + load + risicocheck
- lokale opslag in de browser
- CSV export en JSON backup
- PWA manifest + service worker

## Lokaal draaien

```bash
npm install
npm run dev
```

## Online zetten via Vercel

1. Maak een gratis account op Vercel
2. Upload deze map of koppel GitHub
3. Deploy
4. Open de link op je iPhone in Safari
5. Deelknop → Zet op beginscherm

## Build

```bash
npm run build
```
